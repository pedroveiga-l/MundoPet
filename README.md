# 🐾 Mundo Pet

Sistema de agendamentos para petshop desenvolvido como desafio prático da Rocketseat.

O Mundo Pet permite cadastrar, visualizar e remover agendamentos de atendimentos para pets, organizados por data e período do dia (manhã, tarde e noite).


## 📌 Sobre o projeto

O projeto simula um sistema interno de petshop onde é possível:

✅ Criar um novo agendamento

✅ Remover um agendamento

✅ Visualizar agendamentos por data

✅ Organizar os atendimentos por período do dia

✅ Manipular datas e horários com DayJS

O sistema foi desenvolvido com foco em manipulação de DOM, organização de dados em JSON e controle de datas.


## 🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript (Vanilla JS)

JSON

DayJS

## 🎯 Funcionalidades
### 📅 Visualização de agenda

Seleção de data

Listagem automática dos agendamentos do respectivo dia

Separação por períodos:

🌅 Manhã (09h–12h)

☀️ Tarde (13h–18h)

🌙 Noite (19h–21h)

### ➕ Novo agendamento

O usuário informa:

Nome do tutor

Nome do pet

Telefone

Descrição do serviço

Data

Hora

### ❌ Remoção de agendamento

Possibilidade de excluir um atendimento diretamente da lista


## 🧠 Conceitos aplicados

Manipulação de DOM

Eventos em JavaScript

Organização e filtragem de dados

Manipulação de datas com DayJS

Estruturação de layout com CSS

Separação de responsabilidades

## 🚀 Como executar o projeto

Clone o repositório:

git clone https://github.com/seu-usuario/mundo-pet.git

Abra o projeto no VS Code

Execute o JSON Server no terminal: npm run server

Abra um novo terminal e digite: npm run build
Em seguida digite: npm run dev
